# 🛠️ Hardware Components List

The following components were used to build the Smart Irrigation System. This list is intended to help beginners and students source the correct parts for their project.

| Component Name | Description | Quantity | Link (Reference) |
| :--- | :--- | :--- | :--- |
| **Microcontroller** | ESP8266 (NodeMCU) / ESP32 / Arduino Uno | 1 | [Buy/Info](https://www.adafruit.com/product/2471) |
| **Soil Moisture Sensor** | Capacitive or Resistive Soil Moisture Sensor | 1 | [Buy/Info](https://www.adafruit.com/product/4026) |
| **5V Relay Module** | To control the water pump (On/Off) | 1 | [Buy/Info](https://www.adafruit.com/product/3191) |
| **Submersible Water Pump** | DC 3V-6V Mini Water Pump | 1 | [Buy/Info](https://www.amazon.com/dp/B07R9L7R7Y) |
| **Power Supply** | 5V DC Adapter or Battery Pack | 1 | - |
| **Jumper Wires** | Male-to-Male / Male-to-Female | 10+ | - |
| **Breadboard** | Mini or Half-size Breadboard | 1 | - |
| **Pipe/Tubing** | Small diameter vinyl tubing for the pump | 1m | - |

---

> [!TIP]
> **Pro Tip:** Using a **Capacitive** Soil Moisture Sensor is highly recommended over a resistive one as it doesn't corrode over time when buried in wet soil.
